Language Attributes in Markup

HTML Document
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Document</title>
</head>
<body>
  <h1>Hello World</h1>
  <p>This is an English document.</p>
</body>
</html>
The lang="en" attribute specifies that this HTML document is in English.
